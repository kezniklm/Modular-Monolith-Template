# VertoBank

## Online Banking System

### Official Assignment
Design an online banking system for account management and transaction processing. Accounts can be created in various currencies. Currency exchange is based on user-driven supply and demand.

### Project Overview
VertoBank is a modern online banking platform that enables users to manage accounts, perform transactions, and exchange currencies securely and efficiently.

#### Key Features
- **Multi-currency Accounts:** Users can open accounts in different currencies.
- **User-driven Currency Exchange:** Currency exchange rates are determined by offers and requests between users, reflecting real market supply and demand.
- **Secure Transactions:** All transactions are processed securely, ensuring data integrity and privacy.
- **Account Management:** Users can view balances, transaction history, and manage their accounts easily.

#### Technologies Used
- .NET 10.0
- C#
- Modular architecture for scalability and maintainability

#### Getting Started
1. Clone the repository.
2. Build the solution using your preferred .NET development environment.
3. Configure connection strings and settings in `appsettings.json`.
4. Run the application using the provided launch profiles.


#### Folder Structure
- `VertoBank/` - Main application
- `VertoBank.AppHost/` - Application host (based on Aspire)
- `VertoBank.Modules/` - Modular components (Application, Domain, Infrastructure, Presentation, SharedKernel)
- `VertoBank.ServiceDefaults/` - Service extensions and defaults (based on Aspire)
